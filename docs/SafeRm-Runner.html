runSafeRm :: (MonadFsReader m, MonadFsWriter m, HasCallStack, MonadCallStack m, MonadUnliftIO m, MonadTerminal m, MonadSystemTime m) => m () Source #

Entry point for running SafeRm. Does everything: reads CLI args, optional Toml config, and creates the environment before running SafeRm.

Since: 0.1

getConfiguration :: (HasCallStack, MonadFsReader m, MonadCallStack m, MonadUnliftIO m) => m (TomlConfig, Command) Source #

Parses CLI Args and optional TomlConfig to produce the user configuration. For values shared between the CLI and Toml file, the CLI takes priority.

For example, if both the CLI and Toml file specify the trash home, then the CLI's value will be used.
