toPathData :: (MonadFsReader m, HasCallStack, MonadCallStack m, MonadIO m) => Timestamp -> PathI TrashHome -> PathI OriginalPath -> m PathData Source #

For a given filepath, attempts to capture the following data:

  • Canonical path.
  • Unique name to be used in the trash directory.
  • File/directory type.
